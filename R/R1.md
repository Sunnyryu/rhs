```
What`s R?

R은 객체지향 프로그래밍 언어
R은 통계 분석과 data의 시각화를 소프트웨어 환경
R은 데이터분석에 필요한 최신 알고리즘, 방법론 등의 패키지의 집합이다.
R은 data의 시각화를 위한 다양한 그래픽 도구를 제공!
R은 모든 객체는 메모리로 로딩되어 고속으로 처리되고 재사용 가능
R은 굉장히 범용적으로 사용되고 있음!!
```

```
factor - 동일한 값의 목록을 category로 갖는 벡터 자료
변수가 가질 수 있는 값의 범위로서
예) 성별 변수의 범주는 남, 여
factor는 명목형(Norminal)과 순서형(Ordinal) 유형으로 분류

gender <- c("man","woman", "woman", "man", "man")
plot(gender)  #차트는 수치 데이터만 가능하므로 오류
class(gender) - character
mode(gender) - character

ngender <- as.factor(gender) #범주의 순서가 알파벳 순서로 정렬됨
class(ngender) - factor
mode(ngender) - numeric
table(ngender) #빈도수 반환
plot(ngender) - 차트 생성
is.factor(ngender) - true
ngender    #Levels속성에서 범주를 확인  
man woman woman man woman
Levels: man woman
```

```

Vector : R에서 가진 기본이 되는 자료 구조, 1차원, 선형구조

요소의 접근은 변수[index첨자]로 접근, R에서는 첨자 index는 1부터 시작합니다.
동일한 데이터타입만 저장 가능함
벡터를 생성 함수 : c(), seq(), rep()
벡터 자료 처리 함수 : union(), seldiff(), intersect(), ....

c(1:20)
1:20
c(1,1,2,3,3,3,4,5,5,5,5)
```
