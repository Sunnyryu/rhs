```
일반적 문제

GAN에는 여러가지 공통 고장(실패?) 모드가 있음.

사라지는 경사도 
실제로 최적의 판별기는 발전기가 발전하기 위한 충분한 정보를 제공하지 않음.
(판별기가 너무 좋으면 구배 소멸로 인한 발전기 훈련이 실패할 수 있음.!?)

와서스틴 손실: 와서슈타인 손실은 판별자를 최적의 상태로 훈련시킬 때에도 구배 현상이 사라지는 것을 방지하기 위해 고안
수정된 최소최대 손실: 소멸하는 구배를 다루기 위해 손실을 최소화하기 위한 수정을 제안

mode collapse(모드 붕괴)
GAN이 매우 다양한 출력을 생산하기 원함.

발전기가 특히 그럴듯한 출력을 내는 경우, 발전기는 그 출력만 생산하는 것을 배울 수 있음.
발전기는 항상 판별자에게 가장 그럴듯하게 보이는 하나의 출력을 찾으려고 함.

제너레이터가 동일한 출력(또는 작은 출력 세트)을 반복해서 생산하기 시작하는 경우, 판별자의 최선의 전략은 항상 출력을 거부하는 방법을 배우는 것.

각각의 발전기 반복은 특정 판별기에 대해 지나치게 최적화되며 판별기는 트랩에서 빠져나오는 방법을 결코 배우지 못함.
결과적으로 발전기는 작은 출력 유형 세트를 통해 회전한다. 이런 형태의 GAN 고장은 모드 붕괴

해결 시도

와서스틴 손실: 와세르슈타인 손실은 당신이 판별자를 소멸되는 경사도에 대한 걱정 없이 최적성으로 훈련시킬 수 있게 함으로써 모드 붕괴를 완화, 판별기가 로컬 미니마에 고착되지 않으면 발전기가 안정화되는 출력을 거부하는 방법을 학습 (발전기는 새로운 것을 시도해야함!)

펼쳐진 GANs: 펼쳐진 GAN은 현재 판별자의 분류뿐만 아니라 향후 판별기 버전의 출력까지 통합한 발전기 손실 기능을 사용
발전기는 단일 판별기에 대해 지나치게 최적화할 수 없음.

수렴/융합 실패
GANs는 훈련에 관한 모듈에서 논의된 바와 같이 자주 수렴에 실패

해결 시도
판별기 입력에 노이즈 추가 , 판별자 가중치 벌점
```

```
GAN 변형
선된 GAN 기법과 GAN의 새로운 용도를 계속해서 찾아내고 있음.

진보적인 GANs
진보적인 GAN에서 발전기의 첫 번째 레이어는 매우 낮은 분해능 이미지를 생성하며, 후속 레이어는 세부사항을 추가

비교 가능한 비진보적 GAN보다 더 빨리 훈련할 수 있도록 하며, 고해상도 이미지를 생성

조건부 GAN
조건부 GAN은 라벨이 지정된 데이터 집합에서 훈련되며 생성된 각 인스턴스에 대한 라벨을 지정할 수 있음.

조건부 MNIST GAN은 임의의 숫자를 생성하는 반면, 조건부 MNIST GAN은 GAN이 생성해야 하는 숫자를 지정할 수 있음.
결합 확률 P(X, Y)를 모델링하는 대신에 조건부 GANs는 조건부 확률 P(X | Y)를 모델링

이미지 대 이미지 변환
Image-to-Image 변환 GAN은 이미지를 입력으로 가져와서 다른 속성으로 생성된 출력 이미지에 매핑
자동차 모양에 컬러 블럽이 들어간 마스크 이미지를 촬영할 수 있고, GAN은 사실적인 자동차 디테일로 모양을 채울 수 있음)
이미지 대 이미지 GAN을 훈련시켜 핸드백의 스케치를 찍어 핸드백의 사실적 이미지로 만들 수 있음.

손실량은 일반적인 판별기 기반 손실과 픽셀 위상 손실의 가중 조합으로, 발생기가 소스 이미지에서 이탈할 경우 불이익을 줌.

CycleGAN
CycleGAN은 한 세트에서 다른 세트에 그럴듯하게 속할 수 있는 이미지로 이미지를 변환하는 방법을 배움.
ex) cycleGAN은 Lefthand 이미지를 입력으로 지정할 때 아래의 r8and 이미지를 생성 / 말의 이미지를 가져다가 얼룩말의 이미지로 바꿈.

CycleGAN에 대한 훈련 데이터는 단순히 이미지 세트 2개(이 경우 말 이미지 세트와 얼룩말 이미지 세트) / 시스템에는 영상 사이에 라벨이나 쌍방향 서신이 필요하지 않음.

텍스트-이미지 합성
텍스트 대 이미지 GAN은 텍스트를 입력으로 받아들여 텍스트가 그럴듯하고 설명하는 이미지를 생성
(꽃 이미지는 GAN에 텍스트 설명을 입력하여 생성)
시스템에서 GAN은 작은 클래스 집합에서만 이미지를 생성할 수 있다는 점에 유의

초해상도
초해상도 GAN은 영상의 해상도를 높여 필요한 경우 흐릿한 부분을 채우기 위해 디테일을 추가(아래의 흐릿한 중간 이미지는 왼쪽에 있는 원본 이미지의 다운샘플링 버전)

GAN이 생성한 이미지는 원래 이미지와 매우 유사해 보이지만, 머리띠를 자세히 보면 GAN이 원래부터 별버스트 패턴을 재현하지 않았다는 것을 알 수 있을 것 /다운샘플링으로 지워진 무늬를 대신할 수 있도록 나름대로 그럴듯한 무늬를 구성

얼굴 인페인팅
GANs는 그림 작업에서 의미론적 이미지에 사용되어 왔으며, 인 페인팅 작업에서는 이미지의 덩어리가 검게 지워지고, 시스템은 누락된 덩어리를 채우려고 함.

텍스트 음성 변환
모든 GAN이 이미지를 생성하는 것은 아님.
ex) 연구원들은 또한 텍스트 입력에서 합성 음성을 생산하기 위해 GAN을 사용해왔음.
``` 
