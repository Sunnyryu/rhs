## My Ubuntu

#### Ubuntu 5

```
셀 스크립트

환경변수 값을 변경하려면 export 환경변수= 값 실행

sh 명령으로 실행
  sh 스크립트파일 명령으로 실행 가능

실행 기능 속성으로 변경 후 실행
  ./스크립트파일
(chmod +x 파일명)

변수

셀 스크립트에서는 처음 변수가 설정되면 자동으로 변수 생성
변수에 넣는 모든 값은 문자열로 취급(숫자도 문자 취급)
변수 이름은 대소문자 구분!
변수 대입 시 '='에 좌우 공백은 없어야 함

$ 문자가 들어간 글자를 출력하려면 ''로 묶거나 앞에 \를 해줘야함.

변수로 숫자 계산을 하고 싶다면 expr을 사용해야 하며 괄호를 사용하려면 \(괄호)\를 사용해야함

파라미터 변수
$0, $1, $2 등의 형태로 사용

ex) apt-get -y install abc라는 걸 사용한다고 하면
$0은 apt-get / $1은 -y / $2는 install / $3은 abc임

if 문과 case 문

기본 if문

if [ 조건  ]
then
  참일 경우 실행 (ex: echo "참")
  fi

조건 안에 단어를 쓸 때 : 모두 공백이 있어야 함!

if-else문

if [ 조건 ]
then
참일 경우 실행
else
참이 아닐 경우 실행
fi

조건문에 들어가는 비교 연산자

"문자열1" = "문자열2" (두문자열이 같으면 참)
"문자열1" != "문자열2" (두문자열이 같지 않으면 참)
-n "문자열" (문자열이 NULL이 아니면 참)
-z "문자열" (문자열이 NULL이면 참)

수식1 -eq 수식2 (두 수식(or 변수)이 같으면 참)
수식1 -ne 수식2 (두 수식(or 변수)이 같으면 거짓)
수식1 -gt 수식2 (수식1이 크다면 참)
수식1 -ge 수식2 (수식1이 크거나 같으면 참)
수식1 -lt 수식2 (수식1이 작으면 참)
수식1 -le 수식2 (수식1이 작거나 같으면 참)
!수식(수식이 거짓이라면 참)

파일과 관련된 조건

-d 파일이름 (파일이 디렉토리면 참 )
-e 파일이름 (파일이 존재하면 참)
-f 파일이름 (파일이 일반 파일이면 참)
-g 파일이름 (파일이 set-group-id가 설정되면 참)
-r 파일이름 (파일이 읽기 가능이면 참)
-s 파일이름 (파일 크기가 0이 아니면 참)
-u 파일이름 (파일에 set-user-id가 설정되면 참)
-w 파일이름 (파일이 쓰기 가능 상태면 참)
-x 파일이름 (파일이 실행 가능 상태면 참)

case~esac 문 (다중 분기)

ex)
#!/a/sh
case " 1" in
  start)
  echo "시작--";;
  stop)
  echo "중지--";;
  restart)
  echo "다시 시작--";;
  *)
  echo "뮌지 모름--";;

esac
exit 0

AND OR 연산자 (관계)

and = -a or &&
or = -o or ||
-a나 -o는 테스트문 안에서 사용 가능

for ~ in 문
  for 변수 in a1 a2 a3 a4 ....
  do
    반복할 문장
  done

  (변수 개수만큼 반복)

while 문
조건이 참인 동안에 계속 반복 !
ex)
#!/abc/sh
while [1]
do
echo "123"
done
exit 0

until 문
while 문과 용도가 거의 같지만 until 문은 조건식이 참일 때까지 계속반복

until [ $i -gt 10 ]

break, continue, exit, return

break는 반복문 종료 / continue는 반복문의 조건식으로 돌아감
exit는 해당프로그램 종료 / return은 함수 안에서 사용 가능(함수 호출)

사용자 정의 함수

함수이름 () { /(함수 정의)
  내용

}
함수이름(함수 호출)

함수 파리미터 사용

함수이름 (){ /(함수 정의))

  $1, $2 등을 사용
}
함수이름 파라미터1, 파라미터2 (함수를 호출)

export (외부 변수 선언 - 선언한 변수를 다른 프로그램에서도 사용 가능)

printf(c 언어의 printf() 함수와 비슷하게 형식 지정 출력 가능)

set / $(명령)

리눅스 명령을 결과로 사용하려면 '$(명령)' 혀식을 사용해야함 (결과를 파라미터로 사용하고자 할떄는 set 사용)
```
