- 빅데이터
```
빅데이터는 비정형 중심의 데이터 증가
대용량, 고속의 다양한 데이터를 분석하여 인사이트의 가치를 주는 새로운 기술
페타바이트 이상 / 소셜미디어, 로그파일, 스트리밍, 동영샹 등 비정형 데이터 중심
처리 복잡도가 높고 새로운 기술이 필요 잘 정의된 데이터 모델, 절차 등이 없음 상관관계 중심
```

```
빅데이터의 속성
(Variety, Velocity, Volume, Value)

6V
크기(Volume) : 방대한 양의 데이터(테라, 페타바이트 이상의 크기)
다양성(Varity) : 정형(DBMS, 전문 등) + 비정형(SNS, 동영상, 사진, 음성, 텍스트 등)
속도(Velocity) : 실시간으로 생산되며, 빠른 속도로 데이터를 처리/분석
진실성(Veracity) : 주요 의사결정을 위해 데이터 품질과 신뢰성 확보
시각화(Visualization) : 복잡한 대규모 데이터를 시각적으로 표현
가치(Value) : 비즈니스 효익을 실현하기 위해 궁극적인 가치를 창출

NoSQL 기반의 저장 기술
File System 기반의 저장 기술
클라우드 기반의 저장 기술
네트워크 기반의 저장 기술

성공적인 빅데이터 시스템의 구축과 운영 시 반드시 내재화하고 자산화해야 할 세가지요소
사람
기술
데이터

빅데이터 프로젝트
플랫폼 구축형 프로젝트
빅데이터 분석 프로젝트
빅데이터 운영 및 관리
```

```
하둡

방대한 양을 처리하는 플랫폼 (오픈소스)
코어는 Java, C/C++, Python 등 지원
분산파일시스템(HDFS)
분산병렬처리시스템(MapReduce)
기반소프트웨어프레임워크(Core)

특징

대용량 데이터처리능력 / 장애 허용 / 높은 장애 대응력 / 데이터의 스트리밍 액세스 / 간단한 데이터 일관성 모델

하둡으로 클릭 스트림 데이터/ 콜 센터 데이터/ 이메일과 인스턴트 메세지/ 서버 로그/ 세그먼트 데이터/ 센서 데이터 / 비정형 데이터 / 지리적인 데이터 등을 저장 가능
```
